## 🖥 Projeto

O DevLinks é um agregador de links para usar como cartão de visitas online.

## 🚀 Tecnologias

Esse projeto foi desenvolvido durante o curso Discover da Rocketseat com as seguintes tecnologias:

- HTML e CSS
- JavaScript
- Git e Github
- Figma
